10. კომენტარის სახით ახსენით თუ რა არის id, კლასები, და style inheritance. მოიყვანეთ 1 მაგალითი თითოეულისთვის.

id არის ატრიბუტი, რომელიც ელემენტს ხდის უნიკალურს. როცა გვინდა, რომ კონკრეტულ ელემენტს მივანიჭოთ განსხვავებული სტილი. 
ყველა ელემენტს თავისი ერთადერთი id აქვს. შეგვიძლია CSS-ში id-ს მივწვდეთ # სიმბოლოთი და აიდის სახელით. მაგ:

-> HTML

<p id="star">Marylin</p>

-> CSS

#star {
    color: purple;
}


class ასევე ატრიბუტია, რომელსაც ვიყენებთ, როცა გვინდა რამდენიმე ელემენტის ერთნაირად გაფორმება და თავს არ 
ვიკლავთ ერთი და იმავე კოდის ორასჯერ წერით. განსხვავებით id-სგან, class-ის ერთი და იგივე მნიშვნელობა რამდენიმე ელემენტს
შეგვიძლია მივანიჭოთ. მასთან CSS-ში მისაწვდომად, ვწერთ . წერტილს და კლასის მნიშვნელობას. მაგ:

-> HTML

<h1 class="actress">Audrey</h1>
<h2 class="actress">Meryl</h2>

-> CSS 

.actress {
    font-size: 20px;
    text-align: center;
}


style-inheritance ნიშნავს მემკვიდრეობას, ანუ როცა მშობელი ელემენტისგან ვწვდებით შვილ ელემენტს და ვაფორმებთ.
მაგ: 

-> HTML

<div>
    <p>Broweuli</p>
</div>

-> CSS 

div p {
    border: 2px dashed pink;
    color: beige;
}
