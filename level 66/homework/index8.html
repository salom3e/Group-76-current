How HTML was created

HTML (HyperText Markup Language) was created in 1991 by Tim Berners-Lee. Its main purpose was to share and structure information on the World Wide Web.

Development of HTML

Over time, HTML improved and evolved. Important versions include HTML 2.0, HTML 4.01, XHTML, and the modern HTML5.

HTML5 is currently the most widely used version of HTML.
