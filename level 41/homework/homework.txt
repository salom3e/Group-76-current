1. კომენტარის სახით ჩამოწერე თითეული სიის ფუნქცია რაც ვისწავლეთ და მიუწერეთ გვერდით ახსნა განმარტება თუ რა გადაეცემა 
თითეულს და რას აკეთებს ის


.append() - სიის ამ ფუნქციას აუცილებლად ეძლევა არგუმენტი და არგუმენტად მიწვდილი მნიშვნელობა დაემატება სიას ბოლოში. 

მაგალითად:  
list = [2, 4, "lala" True]
list.append("pizza")
print(list) 
--> list = [2, 4, "lala", True, "pizza"]


.insert() - ამ ფუნქციას ეძლევა 2 არგუმენტი: პირველ არგუმენტად ის ინდექსი, რომელზეც გვინდა რომ ჩავამატოთ მნიშვნელობა,
ხოლო მეორე არგუმენტად - მნიშვნელობა, რაც უნდა ჩავამატოთ ამ ინდექსზე.

მაგ: 
list = [2, 4, "lala" True]
list.insert(2, "pizza")
print(list) 
--> list = [2, 4, "pizza", "lala", True]


.pop() - ამ ფუნქციაში არგუმენტად ჩავწერთ იმ ინდექსს, რომელზე მყოფი ელემენტის "ამოგდება", ანუ ამოშლაც გვინდა სიიდან.

მაგ:
list = [2, 4, "lala" True]
list.pop(3)
print(list) 
--> list = [2, 4, "lala"]


.remove() - სიის ამ ფუნქციას კი არგუმენტად თვითონ იმ ელენეტს ვაწვდით, რომლის ამოშლაც გვინდა. ანუ ინდექსის მაგივრად 
თავად ელემენტს ვწერთ. 

მაგ:
list = [2, 4, "lala" True]
list.remove("lala")
print(list) 
--> list = [2, 4, True]
