13. კომენტარის სახით ჩამოწერე თუ რა არის flex-wrap და რაში გვეხმარება ის, ასევე ჩამოწერე რა მნიშვენლობები გადაეცემა მას
და რომელი მათგანი რაზეა პასუხისმგებელი მოიყვანე სამივეზე თითო მაგალით


flex-wrap - მშობელ ელემენტში სხდებიან შვილი ელემენტები და არ ცდებიან მას, ანუ მასში იფუთებიან.

-------------------------------------------------------------------------------------------------

მნიშვნელობები:

flex-wrap: wrap; - ჩვულებრივ იფუთება მშობელ ელემენტში შვილი ელემენტები.


h1 {
    flex-wrap: wrap;
}

-------------------------------------------------------------------------------------------------


flex-wrap: nowrap; - იგივეა როგორც ჩვეულებრივ, ანუ არ იფუთება მშობელში.


p {
    flex-wrap: nowrap;
}


-------------------------------------------------------------------------------------------------


flex-wrap: wrap-reverse; - უკუღმა ფუთავს შვილს მშობელში, შემობრუნებულად. 


i {
    flex-wrap: wrap-reverse;
}
